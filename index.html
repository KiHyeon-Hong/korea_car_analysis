데이터 클린징

데이터 분석을 위한 data set에서 분석가의 모든 기대를 충족하는 데이터를 없다. 따라서 분석가가 데이터 분석을 위해서 데이터 클린징 과정은 매우 중요하다.
데이터 클린징으로 크게 3가지 데이터를 제거/대체 한다.

  • 결측 데이터
  • 이상 데이터
  • 중복 데이터

본 데이터 분석에서 데이터 클린징은 이상치부터 처리하기로 한다.
만약 결측치를 먼저 처리한다면, 결측값을 평균값이나 중위수로 대체 시 많이 다른 이상치로 인한 잘못된 값으로 대체될 가능성도 있다고 판단했기 때문이다.

count() 메소드로 전체적인 데이터의 갯수를 파악한 후, isnull().sum() 매소드로 결측값이 갯수를 파악한다. 이를 통하여 전체 데이터에서 얼만큼의 결측 데이터가 발생했는지 파악하며, 분석 결과 다음과 같은 방법으로 결측 데이터를 제거하도록 한다.


  • 마력은 평균으로 대체한다
  • 토크와 중량은 데이터가 비교적으로 적으므로, 데이터 분석에 영향이 적을 것이라 판단하여 제거하도록 한다.

  • 가격이 증가 할수록 마력과 토크, 배기량, 중량 모두 증가하는 경향을 보인다

의사결정 결론

최종 분석 결과는 다음과 같다. 해당 데이터 분석에서 본 결과 연비가 높을 수록 마력이 낮아지는 경향이 있다. 그리고, LPG 가스를 사용을 하면 연비와 마력이 모두 낮아지는 경향을 보인다. 또한, 변속기가 수동인 경우 자동인 경우에 비해 가격이 낮아지는 경향을 보인다. 따라서 효율적인 소비를 원한다면 LPG 가스를 사용하지 않는 변속기가 수동인 차가 좋을 것이다.
또한, 변속기는 최고가 소형차일수록 없으며, 저가 대형차일수록 수동인 변속기인것을 보인다.
해당 분석의 최종 결론으로 자동차를 구입할때, 이러한 점을 잘 분석하여 구매를 해야하며, 무작정 연비좋거나, 마력이 좋은 차가 아니라 타협점을 찾아 구매를 해야할 것이다.

차별화 포인트

본 데이터 분석을 하면서 차별화된 분석 포인트는 다음과 같다

  • 결측값 처리보다 이상치 처리를 먼저 실시하여, 데이터에 대한 신뢰를 높임
  • 범주형의 이상치를 제거하기 위해 crosstab()을 이용하여 어떤 group이 있는지 눈으로 확인
  • 데이터 분석 과정의 변주형 변수와 연속형 변수의 구분을 명확히 하기 위해 범주형 변수는 object type으로, 연속형 변수는 int, float로 통일
  • 의미가 불분명한 데이터를 의미가 분명하도록 치환
  • 두 연속형 변수의 상관 관계 분석을 위해 corr() 메소드를 이용
  • 연속형 변수를 범주로 묶어 범주형 데이터 분석을 실시
  • 범주를 구분하기 위하여 describe()의 결과를 사용 </font>
